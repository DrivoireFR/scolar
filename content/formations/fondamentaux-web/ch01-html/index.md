# Chapitre 1 — HTML

## Objectifs pédagogiques

À la fin de ce chapitre, tu sauras :

- Expliquer le rôle du HTML dans le front-end et le fonctionnement général du Web
- Produire un document HTML5 valide (`DOCTYPE`, `head`, métadonnées, `body`)
- Structurer le texte, les listes, les tableaux et les citations de façon appropriée
- Utiliser la sémantique HTML5 (`header`, `nav`, `main`, `article`, etc.)
- Intégrer images, vidéos, audio et iframes accessibles
- Construire des formulaires avec labels et validation native
- Appliquer les premières bonnes pratiques d’accessibilité et de SEO technique

Le **style visuel** (CSS) est traité dans le [chapitre 2 — CSS](/formations/fondamentaux-web/ch02-css).
