# Partie 14 — Intégration pratique

Tu as parcouru **cascade, layout, responsive et bonnes pratiques**. Il est temps de **consolider** sur le **Projet fil rouge** : intégration d’une maquette type blog (Scolar).

## Ce que tu vas faire

- **Modalité A — TP intensif** : trois travaux courts (structure + flex + responsive) à partir du dossier de départ.
- **Modalité B — Projet progressif** : un même projet enrichi en trois étapes (HTML → CSS → responsive).

Les deux modalités valident les **mêmes compétences** ; choisis celle qui correspond à ton rythme.

## Prérequis

- [Chapitre 1 — HTML](/formations/fondamentaux-web/ch01-html) validé ou équivalent.
- Les parties CSS **1 à 13** lues et quiz tentés.

Ouvre les **cartes de modalités** ci-dessous pour les consignes détaillées, puis l’[évaluation du chapitre 2](/formations/fondamentaux-web/ch02-css/evaluation).
