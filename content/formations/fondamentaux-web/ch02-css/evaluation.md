# Évaluation — Chapitre 2 (CSS & intégration)

## Contexte

Ce chapitre clôt l’**intégration HTML + CSS** du parcours « Fondamentaux et intégration Web ». Tu as suivi soit le **TP intensif**, soit le **projet progressif** (Partie 14).

---

## Checklist d’auto-évaluation

### Architecture CSS

- [ ] Fichier(s) CSS organisés (sections ou ITCSS simplifié selon ton niveau)
- [ ] Pas de styles inline massifs (sauf cas exceptionnels documentés)
- [ ] Sélecteurs lisibles, spécificité maîtrisée (éviter `!important` sauf nécessité)

### Mise en page

- [ ] **Flexbox** utilisé pour les alignements unidimensionnels pertinents
- [ ] **Grid** pour les grilles bidimensionnelles (cartes, dashboards simples)
- [ ] Pas de layout tableau pour la page entière

### Responsive

- [ ] Approche **mobile-first** ou justifiée
- [ ] **Media queries** testées sur au moins 3 largeurs
- [ ] Contenu lisible sans scroll horizontal involontaire

### Qualité & accessibilité visuelle

- [ ] **Contrastes** de texte suffisants (objectif WCAG AA quand possible)
- [ ] États **focus** visibles sur éléments interactifs
- [ ] **Images** et médias déjà corrects côté HTML (chapitre 1) + tailles/perf côté CSS

### Validation

- [ ] CSS validé ou sans erreurs bloquantes (validateur W3C CSS)
- [ ] Pas d’erreurs HTML réintroduites après intégration

---

## Soumission

Prépare :

1. Dossier source (`index.html`, `styles.css`, assets)
2. Lien de démo (GitHub Pages, Netlify, etc.) si possible
3. Checklist ci-dessus complétée

Seuil indicatif : **7/10** sur la grille de la modalité (TP ou projet) décrite en Partie 14.
