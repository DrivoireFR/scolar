### Questions du quiz (70% requis pour réussir)

1. **Quelle balise HTML représente une section de contenu thématique ?**
   - `<div>`
   - `<section>` ✅
   - `<main>`
   - `<article>`

2. **Quel sélecteur CSS est le plus spécifique ?**
   - `.classe`
   - `#id` ✅
   - `element`
   - `element.classe`

3. **Flexbox utilise par défaut flex-direction ?**
   - `column`
   - `row` ✅
   - `row-reverse`
   - `wrap`

4. **Quelle unité CSS est relative à la taille de la police du parent ?**
   - `px`
   - `em` ✅
   - `vh`
   - `%`

5. **Quel média query breakpoint mobile-first est recommandé ?**
   - `max-width: 768px`
   - `min-width: 768px` ✅
   - `min-width: 600px` ✅
   - Les deux sont valides
